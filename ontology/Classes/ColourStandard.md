# Colour Standard
#### Class
FIAFcore:ColourStandard
#### Reference
FIAF Moving Image Cataloguing Manual 2.3.4.4
#### Definition
(2.3.4.4) Optionally, if considered relevant, describe the system or process by which colour is fixed on the carrier or as part of the digital encoding, for example, Pathécolor, Technicolor, RGB, etc.
