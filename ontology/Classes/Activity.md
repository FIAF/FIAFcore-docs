# Activity
#### Class
FIAFcore:Activity
#### Reference
FIAF Moving Image Cataloguing Manual 1.4.1.1, 2.4.1.1, 3.3.1.1
#### Definition
(1.4.1.1) This describes the activity or role of the Agent in relation to the moving image Work/Variant (e.g. credit terms). (2.4.1.1) Describes the activity of the Agent(s) to make explicit the relationship(s) between the Agent(s) and the Manifestation. (3.3.1.1) Describes the activity of the Agent to make explicit the relationship(s) between the Agent and the Item.
