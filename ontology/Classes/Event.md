# Event
#### Class
FIAFcore:Event
#### Reference
FIAF Moving Image Cataloguing Manual 1.4.2, 2.4.2, 3.3.2
#### Definition
(1.4.2) An Event characterises occurrences in the lifecycle of a moving image Work or its Variants. (2.4.2) An Event characterises occurrences in the life cycle of a moving image Manifestation. Instances of any Event type can have Agent and Other relationships. (3.3.2) An Event characterises occurrences in the life cycle of a moving image Item.
