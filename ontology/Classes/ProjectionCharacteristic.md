# Projection Characteristic
#### Class
FIAFcore:ProjectionCharacteristic
#### Reference
FIAF Moving Image Cataloguing Manual 2.3.4.2, 3.1.5.9
#### Definition
(2.3.4.2) The projection characteristics of a Manifestation include aspect ratio and aperture or image format. (3.1.5.9) The projection characteristics of a Manifestation/Item include aspect ratio and aperture or image format.
