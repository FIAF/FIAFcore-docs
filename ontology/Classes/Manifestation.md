# Manifestation
#### Class
FIAFcore:Manifestation
#### Reference
FIAF Moving Image Cataloguing Manual 2.0
#### Definition
(2.0) A moving image Manifestation is the embodiment of a moving image Work/Variant. Manifestations usually include all analogue, digital and online media associated with a particular embodiment of a Work/Variant.
