# Carrier
#### Class
FIAFcore:Carrier
#### Reference
FIAF Moving Image Cataloguing Manual 2.3.4.1, 3.1.5.1
#### Definition
(2.3.4.1) Carrier type is the medium on or the encoding format in which the Manifestation is fixed. (3.1.5.1) Carrier type is the medium on or the encoding format in which the Item is fixed.
