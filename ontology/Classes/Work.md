# Work
#### Class
FIAFcore:Work
#### Reference
FIAF Moving Image Cataloguing Manual 1.0.1
#### Definition
(1.0.1) A moving image Work comprises both the intellectual or artistic content and the process of realisation in a cinematographic medium, e.g., what it is called, when it was made, who made it, who was in it, what it is about, etc.
