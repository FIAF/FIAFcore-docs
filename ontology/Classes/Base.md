# Base
#### Class
FIAFcore:Base
#### Reference
FIAF Moving Image Cataloguing Manual 3.1.5.13
#### Definition
(3.1.5.13) The physical material or video format on which the Item is captured, for example, describing the flexible transparent material that supports a film items’ emulsion or a magnetic track, (e.g., acetate, nitrate, CTA, etc.).
