# Has Stock
#### Class
FIAFcore:hasStock
#### Reference
FIAF Moving Image Cataloguing Manual 3.1.5.14
#### Definition
(3.1.5.14) Describes the specific stock/brand on which the Item is captured, for example, Eastman Kodak, Fuji, 3M, etc.

[Item](https://fiafcore.org/ontology/Item)
