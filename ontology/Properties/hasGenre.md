# Has Genre
#### Class
FIAFcore:hasGenre
#### Reference
FIAF Moving Image Cataloguing Manual 1.4.3
#### Definition
(1.4.3) Genre describes categories of Works, characterized by similar plots, themes, settings, situations, and characters. Examples of genres are “westerns” and “thrillers”.

[Variant](https://fiafcore.org/ontology/Variant), [Work](https://fiafcore.org/ontology/Work)
