# Has Extent
#### Class
FIAFcore:hasExtent
#### Reference
FIAF Moving Image Cataloguing Manual 2.3.5, 3.1.5.8
#### Definition
(2.3.5) The concept of extent as applied to Manifestations is the “ideal” logical, physical, or temporal(duration) units and not a description of a real physical object. (3.1.5.8)The actual physical / digital extent is a characteristic of a singular Item, since it can be different for multiple Items exemplifying the same moving image Manifestation.

#### Domain
[Item](https://fiafcore.org/ontology/Item), [Manifestation](https://fiafcore.org/ontology/Manifestation)
#### Range
