# Has Sound Standard
#### Class
FIAFcore:hasSoundStandard
#### Reference
FIAF Moving Image Cataloguing Manual 2.3.4.3.1, 3.1.5.5
#### Definition
(2.3.4.3.1) Describes the technical or proprietary system used to record the sound on a Manifestation/Item, for example, Dolby SR, Dolby Digital, etc. (3.1.5.5) Describes the technical or proprietary system used to record the sound on a Manifestation/Item, for example, Dolby SR, Dolby Digital, etc.
