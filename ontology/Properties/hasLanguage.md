# Has Language
#### Class
FIAFcore:hasLanguage
#### Reference
FIAF Moving Image Cataloguing Manual 1.3.5.1, 2.3.3.1
#### Definition
(1.3.5.1) Record the language(s) by taking the most suitable value(s) from a controlled list of languages. (2.3.3.1) Where possible, record the language(s) using the full form of name, e.g., French, Italian, etc. rather than abbreviations or codes, by taking the most suitable value(s) from a controlled list.

[Language Usage](https://fiafcore.org/ontology/LanguageUsage)
