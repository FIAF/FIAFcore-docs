# Has Colour Characteristic
#### Class
FIAFcore:hasColourCharacteristic
#### Reference
FIAF Moving Image Cataloguing Manual 2.3.4.4, 3.1.5.6
#### Definition
(2.3.4.4) The presence of colour(s), tone(s), etc. in a Manifestation. (3.1.5.6) The presence of colour(s), tone(s), etc. in an Item.

#### Domain
[Item](https://fiafcore.org/ontology/Item), [Manifestation](https://fiafcore.org/ontology/Manifestation)
#### Range
