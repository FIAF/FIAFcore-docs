# Has Extent Value
#### Class
FIAFcore:hasExtentValue
#### Reference
FIAF Moving Image Cataloguing Manual 2.3.5, 3.1.5.8
#### Definition
(3.1.5.8) Having separate numeric fields can facilitate calculations in determining the amount of footage that will be preserved. For digital files, enter the numerical measurement indicating the size of the digital asset’s file(s), in KB, MB, GB, or TB.

#### Domain
[Extent](https://fiafcore.org/ontology/Extent)
#### Range
[float](http://www.w3.org/2001/XMLSchema#float)
