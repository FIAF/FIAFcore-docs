# Has Identifier Text
#### Class
FIAFcore:hasIdentifierText
#### Reference
FIAF Moving Image Cataloguing Manual 1.3.1, 2.3.1, 3.1.1
#### Definition
(1.3.1) Create an unambiguous reference to the Work/Variant using a unique identifier and indicate the type of identifier. (2.3.1) Create an unambiguous reference to the Manifestation, such as a specific standard number issued by an official body (i.e. V-ISAN87 or DOI88), or a system-automated assigned ID number. (3.1.1) Create an unambiguous numerical or alphanumerical reference to the moving image Item, such as a call number, barcode, shelf mark or similar, to uniquely identify the copy.
