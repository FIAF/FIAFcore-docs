# Has Form
#### Class
FIAFcore:hasForm
#### Reference
FIAF Moving Image Cataloguing Manual 1.4.3
#### Definition
(1.4.3) Form describes the format and/or purpose of a Work, e.g., “non-fiction”, “short” and “animation” are particular forms or identifiers.

[Variant](https://fiafcore.org/ontology/Variant), [Work](https://fiafcore.org/ontology/Work)
