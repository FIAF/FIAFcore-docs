# Has Line Standard
#### Class
FIAFcore:hasLineStandard
#### Reference
FIAF Moving Image Cataloguing Manual 3.1.5.19
#### Definition
(3.1.5.19) Describes the number of scan lines which make up the image and indicates the resolution, for example, 405, 525, 1080.
