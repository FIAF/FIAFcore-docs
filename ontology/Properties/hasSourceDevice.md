# Has Source Device
#### Class
FIAFcore:hasSourceDevice
#### Reference
FIAF Moving Image Cataloguing Manual 3.1.5.21
#### Definition
(3.1.5.21) Record the source playback device used in transfer projects. This can be a video deck, film printer, or hardware.
