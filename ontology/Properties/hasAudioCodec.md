# Has Audio Codec
#### Class
FIAFcore:hasAudioCodec
#### Reference
FIAF Moving Image Cataloguing Manual 3.1.5.17
#### Definition
(3.1.5.17) The audio compression standard used in the audio file, whether it is contained in a wrapper with a video file or as a standalone audio file.

[Stream](https://fiafcore.org/ontology/Stream)
