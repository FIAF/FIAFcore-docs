# Has Projection Characteristic Value
#### Class
FIAFcore:hasProjectionCharacteristicValue
#### Reference
FIAF Moving Image Cataloguing Manual 2.3.4.2, 3.1.5.9
#### Definition
(2.3.4.2) The aspect ratio (also known as projection ratio) is the projected image area visible on screen, expressed as a value of width to height (the value of height always being “1”).

[Projection Characteristic](https://fiafcore.org/ontology/ProjectionCharacteristic)
