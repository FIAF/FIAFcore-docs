# Has Element
#### Class
FIAFcore:hasElement
#### Reference
FIAF Moving Image Cataloguing Manual 3.1.4
#### Definition
(3.1.4) Record the nature or function of the moving image Item, describing its place in the photochemical or digital production or duplication process, for example, “original negative,” “duplicate negative,” etc.
