# A Un Codec Video
#### Class
FIAFcore:hasVideoCodec
#### Reference
Le Manuel de catalogage des images animées de la FIAF 3.1.5.16
#### Definition
(3.1.5.16) Il s'agit du standard de compression vidéo utilisé dans l'Item vidéo numérique, par exemple, D10, MP4, etc.
