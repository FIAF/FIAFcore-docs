# A Un Événement
#### Class
FIAFcore:hasEvent
#### Reference
Le Manuel de catalogage des images animées de la FIAF 1.4.2, 2.4.2, 3.3.2
#### Definition
(1.4.2) Un Événement caractérise des occurrences dans le cycle de vie d'une Œuvre ou de ses Variantes. (2.4.2) Un Événement caractérise des occurrences dans le cycle de vie d'une Manifestation. Des instances de tout type d'Événement peuvent avoir des relations d'Agent et d'« Autres ». (3.3.2) Un Événement caractérise des occurrences dans le cycle de vie d’un Item.

#### Domain
[Ítem](https://fiafcore.org/ontology/Item), [Manifestation](https://fiafcore.org/ontology/Manifestation), [Variante](https://fiafcore.org/ontology/Variant), [Œuvre](https://fiafcore.org/ontology/Work)
#### Range
