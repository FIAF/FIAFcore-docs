# A Une Utilisation de Langue
#### Class
FIAFcore:hasLanguageUsage
#### Reference
Le Manuel de catalogage des images animées de la FIAF 1.3.5.2, 2.3.3.2
#### Definition
(1.3.5.2) Enregistrez le type d'utilisation d'une langue (p. ex. parlée, intertitres, sous-titres, etc.) en choisissant le terme le plus approprié à partir d'une liste contrôlée interne ou en vous référant à une liste d'autorité existante. (2.3.3.2) Il est possible que plusieurs langues apparaissent dans la Manifestation, selon la façon dont le contenu se présente. Le type d'utilisation des langues définit la forme sous laquelle la langue est exprimée, par exemple, parlée, chantée, écrite, etc.
