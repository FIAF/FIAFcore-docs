# A Un Texte d'Identification
#### Class
FIAFcore:hasIdentifierText
#### Reference
Le Manuel de catalogage des images animées de la FIAF 1.3.1, 2.3.1, 3.1.1
#### Definition
(1.3.1) Créez pour l'Œuvre/la Variante un identifiant unique et indiquez le type d'identifiant. (2.3.1) Créez une référence unique pour la Manifestation, telle qu'un numéro standard délivré par un organisme officiel (par exemple V-ISAN87 ou DOI88), ou un ID automatiquement généré. (3.1.1) Créez une référence numérique ou alphanumérique unique pour l'Item, telle qu'une cote, un codebarres, une marque sur une étagère ou autre, requis pour identifier de manière unique la copie.
