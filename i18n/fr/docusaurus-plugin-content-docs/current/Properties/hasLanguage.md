# A Une Langue
#### Class
FIAFcore:hasLanguage
#### Reference
Le Manuel de catalogage des images animées de la FIAF 1.3.5.1, 2.3.3.1
#### Definition
(1.3.5.1) Enregistrez le(s) langue(s) en choisissant la valeur la plus juste dans une liste contrôlée. (2.3.3.1) Dans la mesure du possible, enregistrez la ou les langue(s) en utilisant la forme complète, p. ex. français, italien, etc. plutôt que des abréviations ou des codes, en choisissant la ou les valeur(s) les plus appropriées dans une liste contrôlée.

#### Domain
[Utilisation de Langue](https://fiafcore.org/ontology/LanguageUsage)
#### Range
[Langue](https://fiafcore.org/ontology/Language)
