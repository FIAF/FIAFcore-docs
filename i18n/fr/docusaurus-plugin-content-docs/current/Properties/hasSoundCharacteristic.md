# A Une Caractéristique Sonores
#### Class
FIAFcore:hasSoundCharacteristic
#### Reference
Le Manuel de catalogage des images animées de la FIAF 2.3.4.3, 3.1.5.3
#### Definition
(2.3.4.3) Les caractéristiques sonores sont des spécifications techniques relatives à l'emplacement du son sur une Manifestation. (3.1.5.3) Il s'agit des spécifications techniques relatives à la fixation du son sur une Manifestation/un Item.

#### Domain
[Ítem](https://fiafcore.org/ontology/Item), [Manifestation](https://fiafcore.org/ontology/Manifestation)
#### Range
[Caractéristique Sonores](https://fiafcore.org/ontology/SoundCharacteristic)
