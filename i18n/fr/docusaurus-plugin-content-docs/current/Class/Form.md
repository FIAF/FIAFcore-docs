# Forme
#### Class
FIAFcore:Form
#### Reference
Le Manuel de catalogage des images animées de la FIAF 1.4.3
#### Definition
(1.4.3) La forme décrit le format et/ou la caractérisation d'une Œuvre, p. ex. « non-fiction », « court métrage » et « animation » sont des formes ou des identifiants
