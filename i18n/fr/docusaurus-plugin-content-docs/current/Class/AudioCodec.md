# Codec audio
#### Class
FIAFcore:AudioCodec
#### Reference
Le Manuel de catalogage des images animées de la FIAF 3.1.5.17
#### Definition
(3.1.5.17) Il s'agit du standard de compression audio utilisé dans un fichier son, qu'il soit dans un format conteneur avec un fichier vidéo ou qu'il existe en tant que fichier audio distinct.
