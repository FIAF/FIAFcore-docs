# Type de Support
#### Class
FIAFcore:Carrier
#### Reference
Le Manuel de catalogage des images animées de la FIAF 2.3.4.1, 3.1.5.1
#### Definition
(2.3.4.1) Le type de support désigne le support ou le format d'encodage sur lequel la Manifestation est fixée. (3.1.5.1) Le type de support est le support ou le format d'encodage sur lequel l'Item est fixé.
