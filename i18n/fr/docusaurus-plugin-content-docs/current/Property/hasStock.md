# A Une Marque du Support
#### Class
FIAFcore:hasStock
#### Reference
Le Manuel de catalogage des images animées de la FIAF 3.1.5.14
#### Definition
(3.1.5.14) Décrit la marque spécifique du support sur lequel l'Item est fixé, par exemple, Eastman Kodak, Fuji, 3M, etc.
