# A Des Lignes de Balayage
#### Class
FIAFcore:hasLineStandard
#### Reference
Le Manuel de catalogage des images animées de la FIAF 3.1.5.19
#### Definition
(3.1.5.19) Cette notion décrit le nombre de lignes de balayage qui constituent l’image et indique la résolution, p. ex. 405, 525, 1080.
