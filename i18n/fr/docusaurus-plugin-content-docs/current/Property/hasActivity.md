# A Une Activité
#### Class
FIAFcore:hasActivity
#### Reference
Le Manuel de catalogage des images animées de la FIAF 1.4.1.1, 2.4.1.1, 3.3.1.1
#### Definition
(1.4.1.1) Ce chapitre décrit l'activité ou le rôle de l'Agent en relation avec l'Œuvre/la Variante (p. ex. termes utilisés au générique). (2.4.1.1) Ce champ décrit l'activité de l'Agent afin de rendre explicite la ou les relation(s) entre l'Agent ou les Agents avec la Manifestation. (3.3.1.1) Décrit l'activité de l'Agent afin de rendre explicite la ou les relation(s) entre l'Agent ou les Agents avec l'Item.
