# Hat Tonkanalkonfiguration
#### Class
FIAFcore:hasSoundFixation
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 2.3.4.3.2, 3.1.5.4
#### Definition
(2.3.4.3.2) Wenn die Manifestation über Ton verfügt, verzeichnen Sie hier die Spurenkonfiguration (z.B. Mono, Stereo, usw.). (3.1.5.4) Wenn das Exemplar über Ton verfügt, verzeichnen Sie im Feld Tonkanalkonfiguration die Spurenkonfiguration (z.B. Mono, Stereo usw.).
