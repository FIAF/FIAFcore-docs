# Hat Identifikatortext
#### Class
FIAFcore:hasIdentifierText
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.3.1, 2.3.1, 3.1.1
#### Definition
(1.3.1) Erzeugen Sie eine eindeutige Verweisung auf das Werk unter Verwendung eines Identifikators. (2.3.1) Erstellen Sie einen eindeutigen Verweis auf die Manifestation, z.B. eine spezifische, von einer offiziellen Stelle vergebene Standardnummer (z.B. V-ISAN90 oder DOI91) oder eine vom System automatisch vergebene ID-Nummer. (3.1.1) Erstellen Sie einen eindeutigen numerischen oder alphanumerischen Verweis auf das Exemplar, wie z.B. eine Signatur, einen Barcode oder Ähnliches, um das Exemplar eindeutig zu identifizieren.
