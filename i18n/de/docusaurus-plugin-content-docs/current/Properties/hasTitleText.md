# Hat Titeltext
#### Class
FIAFcore:hasTitleText
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.3.2, 2.3.2, 3.1.2
#### Definition
(1.3.2) Mindestens ein Titel wird für das Filmwerk oder die Variante erfasst. (2.3.2) Erfassen Sie mindestens einen Manifestationstitel. (3.1.2) Erfassen Sie ggfs. einen Titel für das Exemplar, falls dieser Titel vom Manifestations- und/oder Werktitel abweicht.
