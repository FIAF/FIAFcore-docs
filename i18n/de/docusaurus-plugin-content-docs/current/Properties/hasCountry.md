# Hat Ein Ursprungsland
#### Class
FIAFcore:hasCountry
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.3.3
#### Definition
(1.3.3) Wenn mehr als ein Ort mit einem Werk bzw. einer Variante verbunden ist, wählen Sie den Ort mit der größten Bedeutung.

#### Domain
[Variante](https://fiafcore.org/de/ontology/Variant), [Werk](https://fiafcore.org/de/ontology/Work)
#### Range
[Ursprungsland](https://fiafcore.org/de/ontology/Country)
