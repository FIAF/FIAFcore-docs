# Hat Wert von Umfang
#### Class
FIAFcore:hasExtentValue
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken
#### Definition
(3.1.5.8) Die Verwendung von separaten numerischen Feldern kann Berechnungen zur Bestimmung des Umfangs des vorhandenen Materials erleichtern. Für digitale Dateien geben Sie die numerische Maßeinheit an, die der Größe der Datei des digitalen Exemplars in KB, MB, GB oder TB entspricht.
