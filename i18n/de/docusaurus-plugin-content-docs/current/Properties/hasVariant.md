# Hat Varianten
#### Class
FIAFcore:hasVariant
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.4.5
#### Definition
(1.4.5) Die Beziehung zwischen einem Werk und einer Variante (z.B. Teil/Teil von) sollte ausgedrückt werden. Die Beziehungen zwischen Werk und Variante werden beschrieben oder veranschaulicht, indem sie mit dem Werk-Identifikator verknüpft werden, durch die Benutzung von Beziehungskennzeichnern (relator terms) oder entsprechend den Gegebenheiten der Datenstruktur der Institution beschrieben und veranschaulicht werden.
