# Hat Standort
#### Class
FIAFcore:hasStorageLocation
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 3.1.6.2
#### Definition
(3.1.6.2) Bei der Beschreibung von Exemplaren sollte der Standort in Form einer Lagerortnummer angegeben werden, um den Zugriff und das Wiederauffinden zu ermöglichen.
