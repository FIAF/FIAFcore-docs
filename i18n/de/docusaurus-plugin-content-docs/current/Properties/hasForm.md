# Hat Gattung
#### Class
FIAFcore:hasForm
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.4.3
#### Definition
(1.4.3) Gattung beschreibt das Format und/oder den Zweck eines Werkes, z.B. sind 'Kurzfilm' und 'Animation' spezifische Formen oder Identifikatoren.

[Variante](https://fiafcore.org/de/ontology/Variant), [Werk](https://fiafcore.org/de/ontology/Work)
