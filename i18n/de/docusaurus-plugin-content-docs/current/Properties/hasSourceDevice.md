# Hat Abspielgerät
#### Class
FIAFcore:hasSourceDevice
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 3.1.5.21
#### Definition
(3.1.5.21) Erfassen Sie das in Digitalisierungsprojekten verwendete Abspielgerät. Dies kann ein Videodeck, ein Filmrecorder oder eine Hardware sein.
