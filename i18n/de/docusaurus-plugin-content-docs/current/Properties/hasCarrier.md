# Hat Trägertyp
#### Class
FIAFcore:hasCarrier
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 2.3.4.1, 3.1.5.1
#### Definition
(2.3.4.1) Der Trägertyp ist das Medium bzw. das Kodierungsformat, auf dem die Manifestation fixiert ist. (3.1.5.1) Der Trägertyp ist das Medium oder das Kodierungsformat, auf dem das Exemplar fixiert ist.
