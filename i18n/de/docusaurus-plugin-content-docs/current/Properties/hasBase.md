# Hat Träger
#### Class
FIAFcore:hasBase
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 3.1.5.13
#### Definition
(3.1.5.13) Der Träger bezeichnet das physische Material oder Videoformat, auf dem das Exemplar aufgenommen wurde, z.B. das flexible, transparente Material, das die Emulsion eines Exemplars oder einer Magnetspur trägt (z.B. Azetat, Nitrat, CTA usw.).
