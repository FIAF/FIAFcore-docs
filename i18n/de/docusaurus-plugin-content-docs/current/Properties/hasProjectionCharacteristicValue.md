# Hat Wert von Informationen zur Projektione
#### Class
FIAFcore:hasProjectionCharacteristicValue
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 2.3.4.2, 3.1.5.9
#### Definition
(2.3.4.2) Das Bildseitenverhältnis (auch als Projektionsverhältnis bezeichnet) ist der projizierte, auf der Leinwand sichtbare Bildbereich, ausgedrückt als Wert von Breite zu Höhe (wobei der Wert für die Höhe immer '1' ist), z.B. 2,34:1, 2,39:1.
