# Hat Zeilennorm
#### Class
FIAFcore:hasLineStandard
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 3.1.5.19
#### Definition
(3.1.5.19) Die Zeilennorm beschreibt die Anzahl der Abtastzeilen, aus denen das Bild besteht, und gibt die Auflösung an, z.B. 405, 525, 1080.
