# Hat Filmmaterial
#### Class
FIAFcore:hasStock
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 3.1.5.14
#### Definition
(3.1.5.14) Filmmaterial beschreibt das spezifische Material bzw. die Firmenmarke, auf dem/der das Exemplar aufgenommen wurde, z.B. Eastman Kodak, Fuji, 3M, usw.
