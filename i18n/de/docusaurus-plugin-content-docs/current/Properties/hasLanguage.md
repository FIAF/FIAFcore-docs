# Hat Sprache
#### Class
FIAFcore:hasLanguage
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.3.5.1, 2.3.3.1
#### Definition
(1.3.5.1) Verwenden Sie eine Werteliste oder einen Thesaurus. (2.3.3.1) Wenn möglich, erfassen Sie die Sprache(n) mit der vollen Namensform, z.B. Französisch, Italienisch usw., und nicht mit Abkürzungen oder Codes, indem Sie den/die geeignetsten Wert(e) aus einer kontrollierten Liste nehmen.
