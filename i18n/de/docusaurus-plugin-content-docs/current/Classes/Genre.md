# Genre
#### Class
FIAFcore:Genre
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.4.3
#### Definition
(1.4.3) Das Genre beschreibt Kategorien von Werken, die sich durch ähnliche Handlungsstränge, Themen, Schauplätze, Szenarien und Charaktere auszeichnen. Beispiele für Genres sind 'Western' oder 'Thriller'.
