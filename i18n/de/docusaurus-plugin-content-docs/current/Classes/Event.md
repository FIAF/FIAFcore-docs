# Ereignis
#### Class
FIAFcore:Event
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken
#### Definition
(1.4.2) Ein Ereignis beschreibt Geschehnisse im Lebenszyklus eines Werks oder seiner Varianten. (2.4.2) Ein Ereignis charakterisiert Stationen im Lebenszyklus einer Manifestation. Instanzen eines beliebigen Ereignistyps können Agent- und 'Andere'-Beziehungen haben. (3.3.2) Ein Ereignis kennzeichnet ein Vorkommnis im Lebenszyklus eines Exemplars.
