# Werk
#### Class
FIAFcore:Work
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.0.1
#### Definition
(1.0.1) Ein Filmwerk umfasst sowohl den intellektuellen oder künstlerischen Inhalt als auch den Entstehungsprozess eines Films, z.B. wie er bezeichnet wird, wann er entstand, wer ihn hergestellt hat, wer mitgewirkt hat, worum es sich dabei handelt usw.
