# Exemplarstatus
#### Class
FIAFcore:Status
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 3.1.5.2
#### Definition
(3.1.5.2) Der Exemplarstatus beschreibt den Aufbewahrungs- oder Zugriffsstatus auf das Exemplar, z.B. Mastermaterial, Archivkopie, Vorführkopie, etc.
