# Funktionen
#### Class
FIAFcore:Activity
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.4.1.1, 2.4.1.1, 3.3.1.1
#### Definition
(1.4.1.1) Dieses Kapitel beschreibt die Funktion oder Rolle des Agenten in Bezug auf das Werk bzw. die Variante. (2.4.1.1) Beschreibt die Aktivität des/der Agenten, um die Beziehung(en) zwischen dem/den Agenten und der Manifestation deutlich zu machen. (3.3.1.1) Beschreibt die Funktionen von Agenten, um die Beziehungen zwischen Agenten und Exemplaren deutlich zu machen.
