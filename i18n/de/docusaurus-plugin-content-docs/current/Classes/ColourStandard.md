# Farbverfahren
#### Class
FIAFcore:ColourStandard
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 2.3.4.4
#### Definition
(2.3.4.4) Optional beschreiben Sie das System oder den Prozess, mit dem die Farbe auf dem Träger oder als Teil der digitalen Kodierung fixiert wird, z.B. Pathécolor, Technicolor, RGB usw.
