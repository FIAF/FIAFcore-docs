# Manifestationen
#### Class
FIAFcore:Manifestation
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 2.0
#### Definition
(2.0) Eine Manifestation ist die physische Verkörperung eines audiovisuellen Werkes (bzw. der Variante eines Werks). Manifestationen umfassen in der Regel alle analogen, digitalen und Online-Medien, die mit einer bestimmten Verkörperung eines Werks/einer Variante verbunden sind.
