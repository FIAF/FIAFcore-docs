# Agenten
#### Class
FIAFcore:Agent
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.4.1, 2.4.1, 3.3.1
#### Definition
(1.4.1) Ein Agent ist als eine Entität definiert, die an der Schöpfung, Realisierung, Kuratierung oder Verwertung eines Werkes bzw. einer Variante beteiligt ist. (2.4.1) Agenten für Manifestationen sind an der Verwertung (Veröffentlichung, Vertrieb, Ausstrahlung), Veröffentlichung, Herstellung oder Bewahrung einer Manifestation beteiligt. Sie tragen die Hauptverantwortung für die Manifestation oder sind für sie von großer Bedeutung. (3.3.1) Ein Agent für Exemplare von Werken ist definiert als eine Entität, die an der Beschaffung, der Beförderung (zum Zwecke der Ausleihe, der Überprüfung, der Lagerung usw.) oder der Bewahrung eines Exemplars beteiligt ist und von der man annimmt, dass er die Hauptverantwortung für das Exemplar trägt oder für dieses von großer Bedeutung ist.
