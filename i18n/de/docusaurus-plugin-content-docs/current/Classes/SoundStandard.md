# Tonsystem
#### Class
FIAFcore:SoundStandard
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 2.3.4.3.1, 3.1.5.5
#### Definition
(2.3.4.3.1) Beschreibt das technische oder proprietäre System, das zur Aufnahme des Tons auf einer Manifestation/Exemplar verwendet wird, z.B. Dolby SR, Dolby Digital usw. (3.1.5.5) Tonsystem beschreibt das technische oder proprietäre System, mit dem der Ton auf einer Manifestation bzw. einem Exemplar aufgezeichnet wurde, z.B. Dolby SR, Dolby Digital usw.
