# Variante
#### Class
FIAFcore:Variant
#### Reference
Das FIAF-Handbuch für die Katalogisierung von Filmwerken 1.0.2
#### Definition
(1.0.2) Eine Variante ist eine Entität, die verwendet werden kann, um jede Änderung inhaltsbezogener Merkmale darzustellen, die jedoch den Gesamtinhalt eines Werkes als Ganzes nicht wesentlich verändert.
