# Tiene Norma de Líneas
#### Class
FIAFcore:hasLineStandard
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 3.1.5.19
#### Definition
(3.1.5.19) Describe el número de líneas escaneadas que estructuran la imágen e indica la resolución, por ejemplo, 405, 525, 1080.
