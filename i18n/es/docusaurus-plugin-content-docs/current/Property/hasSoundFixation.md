#### Definition
(2.3.4.3.2) Si la Manifestación tiene sonido, registre la configuración de la pista (por ejemplo, mono, estéreo, etc.). (3.1.5.4) Si el Ítem tiene sonido, anote aquí la configuración de la pista (por ejemplo, mono, estereo, etc.).
