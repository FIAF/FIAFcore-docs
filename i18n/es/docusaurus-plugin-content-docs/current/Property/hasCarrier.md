# Tiene Portador
#### Class
FIAFcore:hasCarrier
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 2.3.4.1, 3.1.5.1
#### Definition
(2.3.4.1) El tipo de portador es el medio en el que se almacena o se codifica una Manifestación. (3.1.5.1) Se refiere al formato de codificación o el medio en el que se ha fijado el Ítem.
