# Tiene Valor Característico de Proyección
#### Class
FIAFcore:hasProjectionCharacteristicValue
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 2.3.4.2, 3.1.5.9
#### Definition
(2.3.4.2) La proporción dimensional (también conocida como relación de aspecto) es el área visible de una imagen proyectada sobre una pantalla, y se expresa como el valor del ancho por el alturo (el valor de la altura siempre se expresa con el valor “1”).
