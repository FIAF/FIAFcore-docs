# Tiene Forma
#### Class
FIAFcore:hasForm
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.4.3
#### Definition
(1.4.3) Un término de Forma describe el formato y/o propósito de una Obra (por ejemplo, “no-ficción”, “cortos” y “animación” son formas particulares o identificadores
