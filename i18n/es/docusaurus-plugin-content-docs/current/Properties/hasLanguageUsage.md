# Tiene Uso del Idioma
#### Class
FIAFcore:hasLanguageUsage
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.3.5.2, 2.3.3.2
#### Definition
(1.3.5.2) Registre el Tipo de uso de un idioma (por ejemplo: hablada, intertítulos, subtítulos, etc.); tomando el término más adecuado de una lista controlada elaborada en la institución o con referencia a una lista autorizada. (2.3.3.2) Un idioma puede presentarse en más de una forma, dependiendo del contexto: el Tipo de uso del idioma define su expresión, por ejemplo: hablado, cantado, escrito, etc.
