# Tiene Agente
#### Class
FIAFcore:hasAgent
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.4.1, 2.4.1, 3.3.1
#### Definition
(1.4.1) Se define a un Agente como una entidad involucrada en la creación, realización, curaduría, o explotación de una Obra/Variante; a quien se considera que tiene una responsabilidad mayor, o que es de importancia mayor para la Obra/Variante. (2.4.1) Un Agente de Manifestaciones de imágenes en movimiento se define como una entidad que está involucrada en la explotación (estreno, distribución, transmisión), publicación, fabricación o preservación de una Manifestación; a quien se considera responsable o de mayor importancia para la Manifestación. (3.3.1) Un Agente de Ítems de imágenes en movimiento se define como una entidad que está involucrada en la adquisición, los movimientos (para préstamo, inspección, almacenamiento, etc.) o la preservación de un Ítem de imágenes en movimiento; quien se considera que tiene una responsabilidad mayor o de la más extensa importancia, para el Ítem de imágenes en movimiento.
