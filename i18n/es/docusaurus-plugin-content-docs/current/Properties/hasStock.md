# Tiene Stock
#### Class
FIAFcore:hasStock
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 3.1.5.14
#### Definition
(3.1.5.14) Describe el tipo o la marca comercial específica sobre la que se ha capturado el Ítem, por ejemplo, Eastman Kodak, Fuji, 3M, etc.
