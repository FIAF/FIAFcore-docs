# Tiene Característica de Proyección
#### Class
FIAFcore:hasProjectionCharacteristic
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 2.3.4.2, 3.1.5.9
#### Definition
(2.3.4.2) Las características de proyección de una Manifestación incluyen la proporción dimensional y la apertura o formato de imagen. (3.1.5.9) Las características de proyección de una Manifestación/Ítem pueden incluir la proporción dimensional y el formato de imagen o apertura.

#### Domain
[Ítem](https://fiafcore.org/es/ontology/Item), [Manifestación](https://fiafcore.org/es/ontology/Manifestation)
