# Tiene Base
#### Class
FIAFcore:hasBase
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 3.1.5.13
#### Definition
(3.1.5.13) El material físico o el formato de video sobre el que se ha capturado el Ítem, por ejemplo, la descripción del material flexible transparente que soporta la emulsión de la película de un Ítem o una pista magnética (por ejemplo, acetato, nitrato, CTA, etc.)
