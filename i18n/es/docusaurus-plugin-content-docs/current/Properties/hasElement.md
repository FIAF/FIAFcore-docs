# Tiene Elemento
#### Class
FIAFcore:hasElement
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 3.1.4
#### Definition
(3.1.4) Registre la naturaleza o función del Ítem de imágenes en movimiento, describiendo su lugar en la producción fotoquímica o digital o en el proceso de duplicación, por ejemplo, “negativo original”, “negativo duplicado”, etc.
