# Tiene Actividad
#### Class
FIAFcore:hasActivity
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.4.1.1, 2.4.1.1, 3.3.1.1
#### Definition
(1.4.1.1) Este describe la actividad o papel de un agente en relación con la Obra/Variante de imágenes en movimiento (por ejemplo, en los créditos). (2.4.1.1) Describa la actividad del Agente(s) para hacer explícitas las relaciones entre los Agente(s) y la Manifestación. (3.3.1.1) Describe la actividad del Agente al hacer explícita las relaciones entre el Agente y el Ítem.
