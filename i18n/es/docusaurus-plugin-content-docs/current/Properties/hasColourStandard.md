# Tiene Estándar de Color
#### Class
FIAFcore:hasColourStandard
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 2.3.4.4
#### Definition
(2.3.4.4) Optativamente, si se considera relevante, describa el sistema o proceso mediante el cual se fijó el color en el portador o como parte de la codificación digital, por ejemplo, Pathécolor, Technicolor, RGB, etc.
