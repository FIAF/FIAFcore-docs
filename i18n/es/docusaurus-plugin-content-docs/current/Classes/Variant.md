# Variante
#### Class
FIAFcore:Variant
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.0.2
#### Definition
(1.0.2) Una Variante de imágenes en movimiento es una entidad que puede usarse para indicar cualquier cambio en las características relativas al contenido, que no modifican de manera significativa el contenido general de una obra como un todo.
