# Idioma
#### Class
FIAFcore:Language
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.3.5.1, 2.3.3.1
#### Definition
(1.3.5.1) Registre el(los) idioma(s) tomando el(los) valor(es) más apropiados de una lista controlada de idiomas. (2.3.3.1) Cuando sea posible, registre el(los) idioma(s) utilizando la forma completa del nombre, por ejemplo, francés, italiano, etc. en lugar de abreviaturas o códigos.
