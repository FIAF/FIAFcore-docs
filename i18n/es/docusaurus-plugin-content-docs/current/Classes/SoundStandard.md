# Sistemas de sonido
#### Class
FIAFcore:SoundStandard
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 2.3.4.3.1, 3.1.5.5
#### Definition
(2.3.4.3.1) Describa el sistema técnico o el sistema patentado que se ha utilizado para grabar el sonido de la Manifestación, por ejemplo, Dolby SR, Dolby Digital, etc. (3.1.5.5) Describe el sistema técnico utilizado para grabar el sonido en una Manifestación/Ítem, por ejemplo, Dolby SR, Dolby Digital, etc.
