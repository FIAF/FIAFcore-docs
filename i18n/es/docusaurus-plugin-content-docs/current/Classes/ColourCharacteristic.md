# Características de Color
#### Class
FIAFcore:ColourCharacteristic
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 2.3.4.4, 3.1.5.6
#### Definition
(2.3.4.4) Se refiere a la presencia de color(es), tono(s), etc., en una Manifestación. (3.1.5.6) La presencia de color(es), tono(s), etc., en un Ítem.
