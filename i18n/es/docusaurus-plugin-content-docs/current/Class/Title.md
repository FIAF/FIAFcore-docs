# Título
#### Class
FIAFcore:Title
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.3.2.1, 2.3.2.1, 3.1.2.1
#### Definition
(1.3.2.1) Si la Obra, Variante o Manifestación incluye múltiples títulos, relevantes para la institución, tales como un título en otro idioma de una variante doblada o subtitulada; registre las variantes del título y asocie cada título con “Tipo de Título” para diferenciar entre los varios tipos de Títulos. (2.3.2.1) Cuando sea relevante para una institución, si la Manifestación incluye multiples títulos, tales como un título en otro idioma para una variante doblada o subtitulada, entonces, registre los títulos relevantes y asocie un “Tipo de Título” a cada uno de éstos para diferenciarlos. (3.1.2.1) Anote la fuente de información del título.
