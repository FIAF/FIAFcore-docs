# Codificación de Audio
#### Class
FIAFcore:AudioCodec
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 3.1.5.17
#### Definition
(3.1.5.17) Se refiere a la norma de compresión de audio aplicada en el archivo de audio, ya sea que esté contenida en conjunto con el video o como un archivo de audio independiente.
