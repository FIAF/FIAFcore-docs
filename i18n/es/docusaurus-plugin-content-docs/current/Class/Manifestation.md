# Manifestación
#### Class
FIAFcore:Manifestation
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 2.0
#### Definition
(2.0) Una Manifestación de imágenes en movimiento es la materialización de una Obra/Variante. Las Manifestaciones generalmente incluyen todos los medios analógicos, digitales y en línea, asociados con una particular materialización de una Obra/Variante.
