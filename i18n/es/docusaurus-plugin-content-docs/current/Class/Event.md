# Evento
#### Class
FIAFcore:Event
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.4.2, 2.4.2, 3.3.2
#### Definition
(1.4.2) Un Evento caracteriza ocurrencias en el ciclo de vida de una Obra de imágenes en movimiento o sus Variantes. (2.4.2) Un Evento caracteriza las ocurrencias en el ciclo de vida de una Manifestación de imágenes en movimiento. Así, cualquier tipo de Evento pueden tener relaciones con Agente(s) y con “Otra(s) relación(es)”. (3.3.2) Un Evento caracteriza las ocurrencias en el ciclo de vida de un Ítem de imágenes en movimiento.
