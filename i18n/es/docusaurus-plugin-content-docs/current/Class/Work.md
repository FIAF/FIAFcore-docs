# Obra
#### Class
FIAFcore:Work
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 1.0.1
#### Definition
(1.0.1) Una Obra de imágenes en movimiento abarca tanto el contenido intelectual como el contenido artístico y el proceso de realización en un medio cinematográfico, p. ej. cómo se le llama, cuándo fue hecho, quién lo hizo, quién aparece, de qué se trata, etc.
