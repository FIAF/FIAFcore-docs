# Características de Sonido
#### Class
FIAFcore:SoundCharacteristic
#### Reference
Manual FIAF de Catalogación de Imágenes en Movimiento 2.3.4.3, 3.1.5.3
#### Definition
(2.3.4.3) Las características de sonido son especificaciones técnicas relacionadas con la colocación del sonido en la Manifestación. (3.1.5.3) Las especificaciones técnicas relativas a la fijación del sonido en una Manifestación/Ítem.
